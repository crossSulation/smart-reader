Read smarter. Learn faster.

Smart Reader supports PDF, EPUB, and Markdown and uses AI to build a knowledge graph, generate summaries, and help you review key ideas.

Get started View on GitHub

Key features

  • Multi-format support

    Ingest PDF, EPUB and Markdown files with automatic text extraction.

  • AI Knowledge Graph

    Extract entities and relationships across documents to visualize and explore concepts.

  • Knowledge Review

    Auto-generated summaries, Q&A prompts, and spaced-recall helpers to aid learning.

Quick start

Clone the repo and run the frontend and backend locally.

git clone https://github.com/crossSulation/smart-reader.git
cd smart-reader
# frontend
cd frontend && npm install && npm run dev
# backend
cd ../backend && python -m venv .venv && .venv/bin/activate && pip install -r requirements.txt
# run backend (example)
uvicorn app.main:app --reload --port 8000

See the repository for exact directories and scripts; the commands above are examples and may need adjustment.
