Read smarter. Learn faster.

Smart Reader is an AI-powered document reader that ingests PDF, EPUB, and Markdown files, builds a knowledge graph, and uses spaced repetition to help you retain what you learn.

Get started View on GitHub

Key Features

  • Multi-format Reader

    Native PDF viewer with page navigation and thumbnails. EPUB reader with contents sidebar and reading themes. Markdown viewer with heading-based TOC, LaTeX math, Mermaid diagrams, and SMILES chemical structures.

  • AI Agent Chat

    Unified chat interface powered by a LangChain tool-calling agent. Ask questions, get explanations, generate quizzes, create flashcards, search the web, and save notes — all in one conversation. Streaming token output with real-time tool step visualization.

  • Knowledge Graph

    Automatically extract key concepts, terms, and entities from ingested documents. Visualize relationships as an interactive force-directed graph. Click nodes to view details and linked pages. Supports cross-document relationship inference via embedding similarity + LLM.

  • Spaced Repetition Review

    SM-2 algorithm with fuzz factor for optimized flashcard scheduling. Review due cards with "Again / Hard / Good / Easy" ratings. Notes tab for browsing saved annotations. Filter by book or tag. Flashcards auto-generated by the AI agent and persisted for review.

  • Learning Workflow

    Select text → Explain, Translate (Chinese / English), or Save as Note. Link notes to knowledge points. Notes list with inline editing, tags, and page jump. Markdown export for notes and CSV export for flashcards.

  • Reading Experience

    Dark mode with system preference detection. Keyboard shortcuts (arrow keys for page navigation, Ctrl+B for AI panel, Ctrl+Shift+D for theme). Desktop AIPanel with drag-to-resize. Mobile floating AI panel with slide-out overlay.

  • Hybrid AI Architecture

    Route AI tasks to local models (Ollama) or cloud providers (OpenAI, Anthropic, DeepSeek) based on capability and privacy mode. Local embedding with sentence-transformers. Reranking pipeline for retrieval quality. Confidence-gated fallback from local to cloud.

  • Token Billing & Credits

    Track cloud AI token consumption per capability. Credit point system with monthly free allowance. Purchase credit packs. Real-time balance display with low-credit warnings. All local/Ollama calls are free.

  • Desktop App

    Built with Tauri 2.0 for Windows, macOS, and Linux. Native title bar with window controls. Offline file caching via IndexedDB. SQLite for local data storage. Full functionality without a browser.

Quick Start

1. Clone & Install

git clone https://github.com/crossSulation/smart-reader.git
cd smart-reader/frontend && npm install

2. Backend Setup

cd smart-reader/backend
python -m venv .venv && .venv\Scripts\activate
pip install -r requirements.txt

3. Configure Environment

cp .env.dev .env
# Edit .env — set LLM_PROVIDER, LLM_API_KEY, etc.

4. Run

# Web mode
cd backend && python run_dev.py
cd frontend && npm run dev

# Desktop mode
start_desktop.bat

Architecture

Frontend

  • React 18 + TypeScript
  • Vite build system
  • Tailwind CSS + MUI
  • Tauri 2.0 desktop shell
  • react-pdf, epubjs, react-markdown
  • vis-network graph visualization

Backend

  • FastAPI + SQLAlchemy
  • SQLite database
  • LangChain agent
  • sentence-transformers embeddings
  • OpenAI / Ollama / DeepSeek LLM
  • JWT authentication

AI Pipeline

  • Hybrid retrieval (BM25 + vector)
  • Cross-encoder reranking
  • Tool-calling agent (6 tools)
  • Knowledge extraction + relation inference
  • Token counting + credit billing
  • SM-2 spaced repetition scheduling
