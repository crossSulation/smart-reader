# Smart Reader

A multi-format reader with AI-powered knowledge features. Supports PDF, EPUB and Markdown input and provides features such as a knowledge graph and knowledge-review workflows to make reading, indexing, and studying documents faster and smarter.

## Key features
- Read PDF, EPUB, and Markdown files
- AI-driven Knowledge Graph: extract entities and relationships from documents
- Knowledge Review: summaries, questions, and recall prompts generated by the AI
- Cross-format indexing and search across documents
- UI (TypeScript) + backend/processing (Python) components

## Quick start (example)
1. Clone the repo
   git clone https://github.com/crossSulation/smart-reader.git
   cd smart-reader

2. Frontend (TypeScript)
   - Install:
     cd frontend
     npm install
   - Run dev server:
     npm run dev
   - Build:
     npm run build
   - Start:
     npm start

3. Backend / Processor (Python)
   - Create virtualenv:
     python -m venv .venv
     source .venv/bin/activate
   - Install requirements:
     pip install -r requirements.txt
   - Run server (example):
     uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

4. Environment variables
   - Typical vars (add to .env):
     - OPENAI_API_KEY (or MODEL_API_KEY)
     - DATABASE_URL
     - VECTOR_DB_URL or local vector DB settings
   - Replace names with actual variables found in the code/config.

## Example usage
- Upload file (PDF / EPUB / Markdown) via the UI or send it to the ingestion API.
- The ingestion service extracts text, creates embeddings, and updates the knowledge graph.
- Use the UI search or the knowledge-review endpoint to generate summaries, Q&A, and study prompts.
